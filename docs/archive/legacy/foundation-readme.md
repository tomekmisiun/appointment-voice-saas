# Inherited Foundation Reference

This directory preserves documentation inherited from the FastAPI Production
Foundation template. These files are useful for understanding the backend
foundation that Appointment Voice SaaS is built on, but they are not the active
product roadmap, status, or product debt register.

Active Appointment Voice SaaS tracking lives in the repository root:

- `README.md`
- `PROJECT_STATUS.md`
- `ROADMAP.md`
- `TECH_DEBT.md`
- `docs/project/implementation-backlog.md`
- `docs/product/scope.md`
- `docs/architecture/domain-model.md`

Foundation references:

| File | Purpose |
|------|---------|
| `template-project-status.md` | Historical verified state of the inherited FastAPI foundation |
| `template-roadmap.md` | Historical foundation roadmap and completed production-readiness work |
| `template-tech-debt.md` | Historical foundation debt register from the template era |
| `template-freeze-checklist.md` | Historical freeze checklist for using the repository as a backend template |
